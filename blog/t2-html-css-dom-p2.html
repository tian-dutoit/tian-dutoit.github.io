Udacity Reflection Part 2

What is grid based design?

The layout of a webpage is divided into a series of columns and rows (a grid) and this allows you to size and arrange elements in an appealing and easy way.

Whats all the hype about responsive webpages?

There are a range of devices that will be used to view your webpage and these will have different screen sizes and use different web browsers. You need to design your webpage so that it will still be be readable and functional no matter how it is viewed and this involves the layout changing in response to the users device.

What is semantic structure?

Semantic structure is writing your HTML in a way that makes the content easier understand. For instance, using tags that provide meaning and context to the reader instead of more generic tags that will achieve the same outcome (e.g. header instead of div or p). Some initial reading indicates that it is important for SEO as well as ranking places different weights on content within different tags.
